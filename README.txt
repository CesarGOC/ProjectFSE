Se puede descargar el proyecto completo clonándolo a través de línea de comandos
 >	git clone https://github.com/CesarGOC/ProjectFSE.git
- El archivo de la documentación del proyecto se encuentra en su respectiva carpeta, 
esta explica procedimiento de creación de servidor web, librerías necesarias, descripción de los componentes, 
consideraciones, explicación del código implementado, enlace a video demostrativo, etc.
- La carpeta de código normal contiene el programa smart_home.py que se puede ejecutar sin la necesidad de la configuración de servidor web.
- La carpeta de códigos separados contiene los programas divididos por cada implementación dentro del proyecto. 
- La carpeta de códigos web server contiene el proyecto final donde se implementa el servidor web web_smart_home.py.
- El plano de la maqueta se encuentra en su respectiva carpeta.
- Los videos de las cámaras deben estar en la misma carpeta de ejecución de cualquiera de las dos versiones de la smart_home.
- Finalmente, la carpeta de esquemáticos y diagramas se encentran la forma en como se conectan los componentes electrónicos por separado.
